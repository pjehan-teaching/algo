Structure de données

Exercice

Réécrire le programme de calculs binaires en utilisant les nouveaux concepts abordés depuis.

Créer un programme permettant d'effectuer des calculs binaires.

Le programme doit demander à l'utilisateur de saisir 2 nombres binaires (composés uniquement de 0 et de 1).

Le programme doit ensuite afficher le résultat de l'addition des deux nombres.

Le résultat des opérations doit être affiché en binaire et en décimal.

Exercice

Créer un programme permettant de calculer le nombre d'adresses IP disponibles sur un réseau.

Le programme doit demander à l'utilisateur de saisir une adresse IP et un masque de sous-réseau.

Le programme doit ensuite afficher le nombre d'adresses IP disponibles sur le réseau.

Exercice

Créer un programme permettant de réaliser des calculs matriciels.

Le programme doit demander à l'utilisateur de saisir deux matrices.

Le programme doit ensuite afficher le résultat de l'addition des deux matrices.
