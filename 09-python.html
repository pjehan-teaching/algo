Python (pour les dev)

Introduction

Il a été créé par Guido van Rossum en 1991 et est aujourd'hui l'un des langages les plus populaires au monde.

Python est un langage de haut niveau, ce qui signifie qu'il est plus proche du langage humain que du langage machine.

Python est un langage multi-paradigme, ce qui signifie qu'il supporte plusieurs styles de programmation (impératif, orienté objet, fonctionnel...).

Python est un langage de programmation interprété, c'est-à-dire qu'il est exécuté ligne par ligne par un interpréteur.

Commentaires

En Python, les commentaires sont précédés du caractère # : 

# Ceci est un commentaire
print("Hello, World!")

Vous pouvez également utiliser des commentaires multilignes en utilisant des triples guillemets : 

"""
Ceci est un commentaire
sur plusieurs lignes
"""
print("Hello, World!")

Saisi utilisateur

Pour demander une saisie à l'utilisateur, vous pouvez utiliser la fonction input() : 

name = input("Quel est votre nom ? ")
print("Bonjour, " + name)

On remarque que la fonction input() renvoie toujours une chaîne de caractères et que le caractère + permet de concaténer des chaînes de caractères.

Récursivité

La récursivité est une technique de programmation dans laquelle une fonction s'appelle elle-même.

Voici un code en Python qui calcule la factorielle d'un nombre en utilisant une boucle while VS la récursivité :

Boucle while

num = 5
fact = 1

while num > 0:
    fact = fact * num
    num = num - 1

print(fact)

Récursivité

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Collections

En Python, il existe plusieurs types permettant de stocker des collections d'éléments :

  • Listes : collections ordonnées d'éléments
  • Tuples : collections ordonnées et immuables d'éléments
  • Set : collections non ordonnées d'éléments uniques
  • Dictionnaires : collections d'éléments indexés par des clés (key/value pairs)

Listes

En Python, les listes sont des collections ordonnées d'éléments : 

fruits = ["apple", "banana", "cherry"] # Créer une liste
fruits[1] = "orange" # Modifier un élément
fruits.append("pear") # Ajouter un élément
fruits.remove("banana") # Supprimer un élément
print(fruits[1]) # Accéder à un élément
Autres opérations sur les listes

Tuples

En Python, les tuples sont des collections ordonnées et immuables d'éléments : 

fruits = ("apple", "banana", "cherry") # Créer un tuple
print(fruits[1]) # Accéder à un élément

Les tuples sont immuables, ce qui signifie qu'une fois qu'ils sont créés, ils ne peuvent pas être modifiés.

Set

En Python, les sets sont des collections non ordonnées d'éléments uniques : 

fruits = {"apple", "banana", "cherry"} # Créer un set
fruits.add("orange") # Ajouter un élément
fruits.remove("banana") # Supprimer un élément
fruits.add("apple") # ⚠️ Les éléments en double ne sont pas autorisés !!
print(fruits)

Dictionnaires

En Python, les dictionnaires sont des collections d'éléments indexés par des clés (key/value pairs) : 

person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
print(person["age"])

Un dictionnaire peut contenir des éléments de différents types (int, str, list, dict...). Cela peut permettre de représenter des données complexes : 

person = {
    "name": "John",
    "age": 30,
    "children": [
        {"name": "Alice", "age": 5},
        {"name": "Bob", "age": 7}
    ]
}
