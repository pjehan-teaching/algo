Les langages de programmation

Statistiques

Il existe plus de 700 langages de programmation différents.

Le site StackOverflow réalise régulièrement des sondages pour connaître (entre autre) les langages de programmation utilisés par les développeurs

Résultats 2023

Bas / Haut niveau

Bas niveau 🛠️

  • Proche du matériel 💻
  • Abstraction faible 🚲
  • Difficile à lire/écrire 📜
  • Exécution rapide 🐎
  • Exemples : C, C++, Assembleur

Haut niveau 🏗️

  • Proche de l'humain 🧑
  • Abstraction forte 🚗
  • Facile à lire/écrire 📖
  • Exécution lente 🐢
  • Exemples : Python, PHP, JAVA

Paradigmes

  • Déclaratif : Langages permettant de décrire le résultat attendu sans décrire les étapes pour l'obtenir. (SQL, HTML, CSS...)
  • Impératif : Programmes structurés autour de la notion de séquence d'instructions. Les langages impératifs utilisent des variables pour stocker des données et des instructions pour modifier l'état du programme. (Python, C, C++, JAVA, PHP...)
  • Orienté Object : Langages permettant de définir des objets et des classes. (C++, C#, JAVA, PHP...)
  • Fonctionnel : Langages permettant de définir des fonctions et de les utiliser comme des variables. (Haskell, OCaml, Scala...)
  • ...

Compilé ou interprété

  • Compilé : Le code source est transformé en code machine avant l'exécution du programme. (C, C++, JavaScript, JAVA...)
    • JIT (Just In Time) : Le code source est compilé à la volée avant l'exécution du programme. (JavaScript, JAVA...)
  • Interprété : Le code source est exécuté directement par un interpréteur. (Python, PHP...)
